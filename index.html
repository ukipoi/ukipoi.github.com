Redis 初识

简介

Redis 是 Remote Dictionary Server ( 远程数据服务 ) 的缩写，是一个开源的高性能键值对 ( key-value ) 数据库。它以字典结构存储数据，并允许其他应用通过TCP协议读取字典内容。

Redis 支持的键值数据类型有：

  • 字符串类型 ( string )
  • 散列类型 ( hash )
  • 列表类型 ( list )
  • 集合类型 ( set )
  • 有序集合类型 ( zset )

功能

  • 持久化
  • 事物
  • 设置生存时间
  • 排序
  • 消息通知
  • 集群

用途

  • 缓存
  • 数据库
